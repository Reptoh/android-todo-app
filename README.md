# android-todo-app
Simple todo app for android. Eclipse + ADT plugin.

Install:

1. Open Eclipse
2. File > Import(Choose cloned project)
3. Click + in General > Existing Project into Workspace and click Next.
4. Choose directory
5. Click finish
6. Run or Export project
